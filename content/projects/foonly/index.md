The site you're reading right now. Built with [Hugo](https://gohugo.io) and plain CSS — no framework, no JavaScript framework, no build complexity beyond PostCSS for CSS nesting and a bit of fingerprinting.

Goals for this site:

- Fast and lightweight
- Respects browser and OS preferences (dark/light mode, font size, reduced motion)
- Easy to update with new posts and projects
- No tracking, no analytics
