This is a browser-based bookmarks manager that allows you to store a list of bookmarks locally. It's still very much a work in progress, and I'm listing it here to shame myself into working on it more actively.

## What it does?

For now it can just manage a list of links. You can add, edit and delete links, and the data is stored in the browser's local storage.

## What will it do?

Eventually I want to add some more features, like:

- Tags to organize bookmarks.
- Accountless syncing with end to end encryption.
- Whatever else I find out I need once I start using it more actively.
